Learning Management System for Python

For CSCE 274 (Software Engineering), we had to create a LMS that teaches users how to code in a new language. The main focus was not on the learning content, but making a modular system that can be expanded to support more features in the future. We spent a month developing the program as a team and had to create a presentation for the class to watch.

We had to have meetings twice every week with a TA and he guided us on how meetings within a company usually goes. We each had a part to complete for the LMS, which for me I had to deal with the UI and a Facade interface between each each class file.
